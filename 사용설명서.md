# 🏫 자리 배치표 사용설명서

> 학반 자리배치, 이젠 종이와 가위 없이 마우스 클릭 몇 번이면 끝!
> 설치도, 회원가입도, 서버도 없습니다. **HTML 파일 하나면 충분합니다.**

브라우저에서 `index.html` 파일을 더블클릭만 하면 바로 실행돼요. 명렬 붙여넣기 → 랜덤 배치 → 인쇄까지 1분이면 끝납니다.

## 🚀 30초 만에 시작하기

1. **학생 명렬 입력** — 상단의 `👥 학생 편집` 버튼을 누르고 이름을 줄 단위로 입력하거나, 엑셀에서 그대로 복사해 붙여넣으세요.
2. **자리 설정** — 왼쪽 사이드바에서 **열 수**(가로)와 **행 수**(세로)를 조정하면 자리 배치가 자동으로 그려져요.
3. **🔀 전체 랜덤** 버튼 클릭 → 자리 배치 완성!
4. **🖨️ 인쇄** 버튼으로 학반·담임을 입력하고 출력하면 끝.

## 📝 학생 명렬 입력하기 — 3가지 방법

`👥 학생 편집` 버튼을 누르면 모달이 열려요. 상단의 탭 3개 중 편한 방식을 고르면 됩니다.

### ① 표 입력 — 한 명씩 차근차근
직접 표에 한 줄씩 입력합니다. **이름만 적어도 되고, 학번은 비워두면 `01, 02, 03 …`이 자동으로 부여**돼요.

> 💡 학번 컬럼이 거추장스럽다면 학번 칸 헤더(`학번 ▼`)나 안내문의 `🔢 학번 입력 표시/숨김` 링크를 눌러 숨길 수 있어요.

### ② 붙여넣기 — 엑셀에서 가장 빠른 방법 (추천!)
엑셀의 이름 열을 통째로 선택해서 `Ctrl+V`(맥은 `Cmd+V`)로 붙여넣기. 이게 제일 빠릅니다.

```
홍길동
김철수
이영희
```

학번까지 같이 넣고 싶다면 탭이나 쉼표로 구분:

```
3101	홍길동
3102	김철수
3103	이영희
```

### ③ 파일 업로드 — CSV / TXT
미리 만들어둔 명렬 파일이 있다면 드래그&드롭으로 끌어다 놓거나 클릭해서 선택. **`.csv`, `.txt` 모두 지원**합니다. 엑셀이라면 `다른 이름으로 저장 → CSV(쉼표로 분리)`로 저장 후 업로드하세요.

입력이 끝나면 `✅ 적용 · 저장`을 눌러야 반영됩니다.

## 🪑 자리 조작 — 마우스와 키보드

### 클릭 한 번 — 선택
자리를 클릭하면 노란색으로 강조되며 선택돼요.

### 두 번 클릭 — 교체
선택된 자리가 있을 때 다른 자리를 더블클릭하면 두 학생이 자리를 바꿉니다.

### 🖐️ 드래그&드롭 — 가장 직관적
자리를 마우스로 끌어서 다른 자리 위로 놓으면 바로 교체. 가장 직관적이에요.

### ⌨️ 방향키 — 한 칸씩 이동
자리를 선택한 상태에서 **↑ ↓ ← →** 를 누르면 인접한 자리와 교체돼요. 미세 조정에 편합니다.

### 📌 우클릭 / 고정 버튼 — 자리 고정
- 자리에서 **우클릭**하면 즉시 고정/해제 토글
- 자리를 선택한 뒤 헤더의 `📌 고정` 버튼을 눌러도 됩니다
- 고정된 자리는 핀(📌) 아이콘이 표시돼요

## 🎲 랜덤 배치 — 두 가지 모드

### 🔀 전체 랜덤
모든 학생을 한 번에 새로 섞습니다. 고정된 자리도 무시하고 다 섞여요.

### 📌 고정 제외 랜덤 ← **이게 핵심!**
고정해둔 자리는 그대로 두고 **나머지만 다시 섞는** 모드.
사용 시나리오:
- 시력이 안 좋은 학생은 앞자리에 **고정** → `📌 고정 제외 랜덤`
- 짝꿍이 정해진 학생들은 **두 자리를 고정** → 나머지만 섞기
- 마음에 드는 배치의 일부만 살리고 싶을 때

### 🔓 전체해제
고정해둔 자리를 한 번에 모두 해제합니다.

### ↩️ 실행취소
직전 작업을 되돌립니다. **랜덤·이동·교체·고정 모두 되돌릴 수 있어요.** 실수해도 부담 없이!

## 👥 짝 구분 표시

왼쪽 사이드바의 `👥 짝 구분 표시` 토글을 켜면 **2명씩 묶어 같은 색**으로 표시돼요. 짝 활동 수업이나 모둠 구성을 미리 그려볼 때 유용합니다.

---

## ⚙️ 좌석 수 조정

왼쪽 사이드바 `🏫 좌석 설정`에서:
- **열 수 / 행 수** — 숫자 입력으로 자유롭게 조정 (최대 12×12)
- **＋ 버튼** — 빈 자리 추가 (홀수 명일 때 유용)
- **－ 버튼** — 마지막 빈 자리 제거

학생 수 > 좌석 수가 되면 ⚠️ 경고가 자동으로 떠요.

## 🖨️ 인쇄 — 테마와 폰트 고르기

상단의 `🖨️ 인쇄` 버튼을 누르면 인쇄 미리보기 모달이 열립니다.

### 학반 / 담임 입력
칠판 아래에 인쇄될 학반(예: `1학년 6반`)과 담임 이름을 적어주세요. 메인 화면의 입력칸과 자동으로 동기화돼요.

### 🎨 인쇄 테마 5종
- **클래식** — 따뜻한 노란빛, 기본 테마
- **모던 미니멀** — 깔끔한 회색 톤
- **캐릭터** — 연두·하늘·핑크의 산뜻한 파스텔
- **클로드톤** — 베이지와 코랄 조합
- **레트로** — 빈티지한 베이지·브라운

### 🔤 폰트 6종
Noto Sans · Pretendard · Gaegu · 고운돋움 · Cafe24 · 나눔 펜체 중에서 선택. 미리보기에서 글씨 모양을 확인할 수 있어요.

설정이 끝나면 `🖨️ 인쇄 시작` → 브라우저 인쇄 창에서 `PDF로 저장`을 골라 파일로 남길 수도 있습니다.

## 💾 자동 저장 · 백업

### 자동 저장
명렬을 입력하거나 자리를 바꾸면 **즉시 브라우저에 자동 저장**돼요. 다음에 같은 브라우저로 열면 그대로 복원됩니다. 새로고침해도 안 사라져요.

### 📤 내보내기 — JSON 백업
`📤 내보내기` 버튼을 누르면 `자리배치_20260521.json` 같은 파일이 다운로드돼요. 명렬, 자리, 고정 정보, 테마까지 전부 들어 있습니다.

### 📥 불러오기 — 다른 PC에서도 동일하게
다른 컴퓨터에서 작업을 이어가거나 동료 선생님과 공유할 때, `📥 불러오기`로 JSON 파일을 선택하면 됩니다.

> 💡 **학기 초·중간·말 백업 추천!** 학기 시작할 때, 자리 바꿀 때마다 한 번씩 내보내두면 안전합니다.

## 🧠 활용 팁

- **시력·청력이 약한 학생** → 앞자리 고정 후 `고정 제외 랜덤`
- **장난기 많은 짝꿍** → 두 학생을 멀리 떨어뜨려 두고 따로 고정
- **수업 분기마다 재배치** → 매번 새로 짜지 말고 이전 JSON을 불러와 일부만 바꾸기
- **학부모 공개수업** → 인쇄 테마 `클래식`이나 `클로드톤`으로 단정하게 출력
- **명렬 가져오기** → 학교 NEIS에서 받은 엑셀의 이름 열을 그대로 복사해 붙여넣기

## ❓ FAQ

**Q. 데이터가 서버로 전송되나요?**
A. 아니요. 모든 정보는 **브라우저 안에서만** 처리되고 저장돼요. 네트워크 없이도 작동합니다.

**Q. 모바일에서도 쓸 수 있나요?**
A. 표시는 가능하지만 드래그&드롭과 우클릭 기반이라 **PC 사용을 권장**합니다.

**Q. 명렬을 잘못 지웠어요!**
A. 즉시 `↩️ 실행취소`를 누르세요. 또는 미리 `📤 내보내기` 해둔 JSON 파일이 있다면 `📥 불러오기`로 복구.

**Q. 짝을 3명씩 묶고 싶어요.**
A. 현재는 2명 단위 짝 표시만 지원해요. 3명 묶음이 필요하면 자리 배치 자체를 그렇게 구성하는 방식으로 사용해주세요.

**Q. 인쇄했을 때 한 페이지에 안 들어가요.**
A. 브라우저 인쇄 설정에서 **여백 = 좁게**, **배율 = 자동** 또는 90~95%로 조정해보세요.
